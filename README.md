# 3d_interact

## Files
./src 소스 파일 저장 공간   
./include 헤더 파일 저장 공간   
./test_field 테스트 공간   
./test_results_log 테스트 결과 저장 공간(mp4, txt)   

## Introduction
A sloppy project first started in college.   
Used PointCloudLibrary   

Transform hand movements into keyboard or mouse values and pass input to the computer.   

## Requirements
~~~
xdotool
pointcloudlibrary
~~~
